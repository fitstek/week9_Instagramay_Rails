== Description


A clone of the Instagram website done in Rails. A user can sign_up, upload and share photos, like photos, buy photos. 

== Technologies Used


* RoR
* Websockets
* Stripe API
* Google maps API
* AJAX
* Bootstrap 
* Capybara for feature testing etc..

== How to use


* git clone git@github.com:fitstek/week9_Instagramay_Rails.git
* cd week9_Instagramay_Rails

* I have uploaded this on Heroku and it is at http://instagramfitstek.herokuapp.com/
